# 尔茄（"小红帽"）角色总结

## 一句话概括
一个用沉默的观察来理解世界、用一顶魔力兜帽来守护自己和他人的"安静的观察者"——她不是天生的英雄，而是在一次次被需要时走出去、最终走回自己内心的人。

## 诞生出的完整形象

尔茄是一个从"被留下"开始的角色。父母早逝，沉默寡言的奶奶将她抚养到7岁后，把她留在了一个陌生村子，只留下一句"你在这里会安全"和一件红色兜帽。这构成了她人格的底层逻辑：学会不去依赖、不去期待、自我消化一切。

她在村子的角落里长大，是一个"安静的观察者"——不主动说话，却默默记住每个人的习惯和细节。这种特质让她后来在冒险中成为队伍的"大脑"，不靠武力而靠细心和巧劲解决问题。但也正是这种"独自承担一切"的模式，在一次冒险事故中引爆了危机：她因为不愿开口说"我不确定"，导致最在乎的伙伴身受重伤。

这个转折让她被迫正视自己的核心矛盾：她一直在帮别人，却从不让任何人帮自己。最终她选择回到村子教书——不是退缩，而是终于看清了自己真正的方向：比起征服世界，她更想帮别人成长。

她的红色兜帽贯穿始终：从奶奶的遗物，到迷雾森林中的救命微光，再到她展开给一个同样安静的小孩看的传承象征——兜帽的意义一直在生长，就像她本人一样。

## 关键特质
- **性格**：安静、观察力强、温柔而倔强、先人后己
- **核心矛盾**：习惯独自承担一切，难以依赖他人——在经历事故后走向和解
- **标志物**：注入魔力后会展开的红色兜帽（月光下发出指引的微光）
- **人生轨迹**：被留下 → 安静观察 → 为救人而出发 → 在冒险中成长 → 因事故觉醒 → 回归教书传承

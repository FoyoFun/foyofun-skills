# OC 角色概要

## 核心印象
一个习惯站在人群边缘的旁观者——白天的图书管理员，夜晚的情绪收藏家。

## 性格底色
- 天生的安静孤独：存在于人群中却又独立于人群之外，不是不合群，而是骨子里的"旁观者"心态
- 敏锐的观察力：能捕捉到别人忽略的情绪碎片——一个人嘴角的微动、一个未尽的眼神
- "不主动但会回应"的温柔：从不会擅自介入他人生活，但如果被真心求助，会给出基于深厚观察的细腻回应
- 核心矛盾：渴望被理解，却又害怕被真正看穿；擅长看透别人，却把自己的情绪藏得很深

## 行为模式
- 在公共场合习惯性观察陌生人，在心里编织他们的故事
- 夜间写作：把白天捕捉到的瞬间情绪展开成短篇文字，写完即收，从不回看，从不示人（"像把念头存进罐子里"）
- 被问到"你还好吗？"会本能地回答"还好"，即使那天其实很糟糕
- 如果有人试图真正走进ta的内心，第一反应是慌乱和躲避

## 外貌特征
- 整体不起眼，容易被忽略——不是特别好看也不是特别丑
- 但有一双安静、仿佛在吸收一切的眼睛
- 手指修长好看（与书籍长年打交道）
- 穿深色宽松衣服，头发略长，偶尔遮住眼睛——像要把自己藏起来

## 背景与成因
- 童年因父母工作频繁搬家，每次刚交到朋友就要离开
- 慢慢养成了"不投入太多"的相处模式——不是创伤，而是自然的适应
- "旁观者"成为了最舒服、最自然的姿态
- 家庭关系和睦，父母对ta很好，没有戏剧性的冲突

## 生活方式
- 职业：在某座城市一条安静老街上的图书馆做管理员
- 白天在书架间整理别人的故事，夜晚书写自己的观察
- 世界背景：现代都市（暂定）

# 洛琳 — 角色概念总结

## 基本档案

- **姓名**：洛琳
- **性别**：女
- **年龄**：二十出头
- **种族**：人类
- **身份**：流浪剑客 / 除魔人 / 星辰剑盟唯一幸存者

## 外貌

银白色长发（因剑盟毁灭事件中的魔法冲击而变白）、琥珀色猫瞳、身材偏瘦但带有肌肉线条、左前臂有一道明显的疤痕。气质清冷疏离，给人以距离感。

## 性格

- **表层**：冷静寡言，不轻易表露情绪，与人保持距离
- **内层**：善良且有正义感，看到不公会忍不住出手；关心人但表达方式生硬笨拙
- **核心缺陷**：习惯独自承担一切、难以信任他人、带有深重的幸存者愧疚

## 背景

从小在隐秘组织「星辰剑盟」中长大，无父无母，视师门为家。剑盟主张剑术与魔法一体修炼，隐居深山，偶尔下山助人。某夜因内部成员背叛而遭外部势力袭击毁灭，洛琳是唯一的幸存者——师父在最后时刻让她提前离开。她的白发和左臂伤疤均是那一夜留下的印记，而内心的伤至今未愈。

## 能力

- **剑术**：剑盟真传，剑招带有星光效果（"星光剑气"）
- **魔法**：风系（加速移动）和火系（剑刃附魔）的基础战斗魔法，以辅助剑术为主
- **武器**：师父留下的遗物之剑，可传导魔法能量

## 人际关系

- **师父（已故）**：中年女性，严厉而温柔，是洛琳在剑盟中最亲近的人，临终前嘱咐她"保护好自己能保护的人"
- **药师同伴（男性）**：年轻流浪药师，性格开朗话多，与洛琳相反。在一次委托任务中结识后死皮赖脸地跟着她旅行，是目前唯一让洛琳愿意接近的人
- **背叛者（身份未定）**：导致剑盟毁灭的内部叛徒，悬而未决的谜题

## 核心目标与冲突

- **表面目标**：追查剑盟毁灭的真相，找出叛徒
- **深层需求**：在失去一切后，寻找新的"活着的意义"
- **内在冲突**：复仇无法填补内心的空洞，但新的方向尚未找到；幸存者愧疚驱使她过度牺牲自我、难以接受善意

## 日常细节

- 夜间因噩梦难眠，常在月下独自练剑
- 不会做饭但嘴硬，每次都把同伴做的饭吃光
- 每到一个新地方会捡一块小石子，作为"这个地方我来过，我还活着"的仪式
- 因被人说眼睛像猫而不太喜欢猫，但猫偏偏总爱粘着她
- 左臂旧伤在雨天隐隐作痛，从不当回事却总被同伴察觉

## 故事潜力

追查叛徒与毁灭真相构成主线悬念；与药师同伴"冰与火"的羁绊成长；从"为过去而活"到"为当下而活"的内心转变弧线；幸存者愧疚的逐步化解。
